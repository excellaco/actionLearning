# Simple Java Application

This is a basic Java application with a single entry point in `src/Main.java`.

## How to Run

1. Open a terminal in the project root directory.
2. Compile the Java file:
   
   ```powershell
   javac src/Main.java
   ```
3. Run the program:
   
   ```powershell
   java -cp src Main
   ```

You should see the output:

```
Hello, world!
```
